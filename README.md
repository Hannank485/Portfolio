This repository contains the source code for my personal portfolio website, built primarily for job applications. The site focuses on clean design, responsive layouts, and smooth interactions to present my work and skills clearly.

## About
The portfolio is designed to be simple, and user-friendly, with an emphasis on thoughtful UI decisions and subtle animations rather than heavy content. It serves as a central place to explore my background, skills, and selected work.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router

## Focus Areas
- Responsive design across devices
- Smooth and purposeful animations and transitions
- Clean component structure
- Attention to spacing, typography, and layout
- Accessible and intuitive navigation

## Deployment
The site is deployed on **Vercel** for reliable hosting and fast performance.
